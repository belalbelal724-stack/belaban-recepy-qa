# 📲 دليل تحويل الداشبورد لتطبيق حقيقي

## 🎯 خطوات مختلفة حسب الجهاز

---

## 1️⃣ على الموبايل Android (Chrome) — الأسهل

### الطريقة A: من المتصفح مباشرة
1. افتح لينك Vercel في **Chrome**
2. اضغط على القائمة (⋮) في أعلى يمين Chrome
3. ستجد خيار **"Add to Home screen"** أو **"Install app"**
4. اضغطه → اكتب اسم التطبيق → **Add**
5. ✅ ستظهر أيقونة على الشاشة الرئيسية مثل أي تطبيق

### إذا لم يظهر خيار التثبيت:
- تأكد أن Chrome محدّث لآخر إصدار
- زر الموقع لمدة 30 ثانية على الأقل
- قد يحتاج زيارتين للموقع قبل ظهور الخيار

---

## 2️⃣ على iPhone / iPad — يجب استخدام Safari

⚠️ **مهم: لازم Safari وليس Chrome**

1. افتح لينك Vercel في **Safari**
2. اضغط زر المشاركة في الأسفل: 📤 (مربع مع سهم)
3. اسحب لأسفل واختر **"Add to Home Screen"** / **"إضافة إلى الشاشة الرئيسية"**
4. اكتب الاسم → **Add**
5. ✅ ستظهر الأيقونة على الشاشة الرئيسية

---

## 3️⃣ على اللاب توب — أسهل طريقة

### Windows (Chrome/Edge):
1. افتح لينك Vercel
2. **يمين شريط العنوان** ستجد أيقونة شاشة صغيرة بسهم ⊕
3. اضغطها → **Install**
4. ✅ سيظهر التطبيق في قائمة Start وعلى سطح المكتب

### Mac (Chrome/Safari):
1. **Chrome:** نفس خطوات Windows
2. **Safari:** File → Add to Dock

---

## 4️⃣ بديل أقوى — TWA (تطبيق Android حقيقي .apk)

إذا تريد ملف **.apk** للتثبيت المباشر مثل واتساب:

### استخدم خدمة PWA Builder المجانية من Microsoft:
1. اذهب إلى: **https://www.pwabuilder.com**
2. ضع لينك Vercel الخاص بك
3. اضغط **Start**
4. سيُولّد لك ملف **.apk** للأندرويد و**.exe** للويندوز و**.app** للماك
5. حمّل الملف وثبّته مباشرة كأي تطبيق

### هذه الطريقة تعطيك:
- ✅ ملف APK حقيقي ترسله للموظفين عبر واتساب
- ✅ يثبت بضغطة واحدة كأي تطبيق
- ✅ أيقونة على الشاشة الرئيسية
- ✅ يفتح مستقل بدون متصفح
- ✅ يعمل أوفلاين بالكامل

---

## ⚡ الحل الأسرع لكل الموظفين

### الخطوة 1: ارفع الموقع على Vercel
- ادخل vercel.com وسجل بحساب GitHub
- ارفع مجلد rasbi-dashboard على GitHub repo
- في Vercel اربط الـ repo واضغط Deploy
- ستحصل على لينك مثل: `rasbi-qatar.vercel.app`

### الخطوة 2: ولّد ملفات التطبيق
- ادخل **pwabuilder.com**
- ضع اللينك واضغط Start
- حمّل **Android Package** (سيعطيك .apk)
- حمّل **Windows Package** (سيعطيك .msix)

### الخطوة 3: وزّع على الموظفين
- أرسل ملف **.apk** عبر واتساب → الموظف يضغط عليه → يثبت كأي تطبيق
- ملف **.msix** للابتوب → نقرة واحدة وينصب
